# stack-overflow-survey-analysis
Data analysis project based on the Stack Overflow Developer Survey dataset using Python, Pandas, Matplotlib and Seaborn.
# Stack Overflow Developer Survey Analysis

## Overview
This project explores the Stack Overflow Developer Survey dataset using Python and Pandas.

## Objectives
- Analyze developer demographics
- Explore Python popularity
- Investigate remote work trends
- Analyze salary distribution
- Create visualizations and insights

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Visualizations
- Work Experience Distribution
- Remote Work Analysis
- Python Popularity by Age Heatmap
- Salary Analysis by Country
